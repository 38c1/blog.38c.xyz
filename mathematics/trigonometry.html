Defining Points

To start, you need to define the points of the triangle. For consistency, variables the user defines will be lowercase and all other variables will be uppercase.

Define them as \(a\), \(b\), and \(c\).

\[a=(0,0)\] \[b=(0,0)\] \[c=(0,0)\]

Ensure these points are draggable in the editor.

Creating the Triangle

Create a polygon. This will be the outline of the triangle.

\[polygon(a,b,c)\]

Optionally, you can also add the centroid with the following calculation.

\[mean(a,b,c)\]

Incircle Calculations

We can now calculate the incircle. We need to calculate the incircle points first.

Define them as \(A_i\), \(B_i\), and \(C_i\).

\[A_{i}=\sqrt{\left(\left(b.x-c.x\right)^{2}\right)+\left(\left(b.y-c.y\right)^{2}\right)}\] \[B_{i}=\sqrt{\left(\left(a.x-c.x\right)^{2}\right)+\left(\left(a.y-c.y\right)^{2}\right)}\] \[C_{i}=\sqrt{\left(\left(a.x-b.x\right)^{2}\right)+\left(\left(a.y-b.y\right)^{2}\right)}\]

With these calculations, you can calculate the radius of the incircle as well as the position.

For the position of the incircle, we will define it as \(i\).

\[i=\left(\frac{\left(A_{i}\cdot a.x\right)+\left(B_{i}\cdot b.x\right)+\left(C_{i}\cdot c.x\right)}{A_{i}+B_{i}+C_{i}},\frac{\left(A_{i}\cdot a.y\right)+\left(B_{i}\cdot b.y\right)+\left(C_{i}\cdot c.y\right)}{A_{i}+B_{i}+C_{i}}\right)\]

For the radius of the incircle, we can define \(r_i\) as the following.

\[r_{i}=\sqrt{\frac{\left(\frac{A_{i}+B_{i}+C_{i}}{2}-A_{i}\right)\left(\frac{A_{i}+B_{i}+C_{i}}{2}-B_{i}\right)\left(\frac{A_{i}+B_{i}+C_{i}}{2}-C_{i}\right)}{\frac{A_{i}+B_{i}+C_{i}}{2}}}\]

Now we can display the incircle.

\[\left(x-i.x\right)^{2}+\left(y-i.y\right)^{2}=r_{i}^{2}\]

Circumcircle Calculations

Now, the circumcircle. Arguably the most challenging part in trigonometry. Try to follow along.

The position of the circumcircle is the intersection of two of the centerlines. The size of the circumcircle is the size in which the diameter touches \(a\), \(b\), and \(c\).

Firstly, we will calculate \(A_k\), \(B_k\), and \(C_k\).

\[A_{k} =\sqrt{\left(b.x-a.x\right)^{2}+\left(b.y-a.y\right)^{2}}\] \[B_{k} =\sqrt{\left(c.x-b.x\right)^{2}+\left(c.y-b.y\right)^{2}}\] \[C_{k} =\sqrt{\left(a.x-c.x\right)^{2}+\left(a.y-c.y\right)^{2}}\]

Then we can calculate the circles radius as \(r_k\).

\[r_{k}=\frac{A_{k}B_{k}C_{k}}{4\sqrt{\frac{\left(A_{k}+B_{k}+C_{k}\right)}{2}\left(\left(\frac{\left(A_{k}+B_{k}+C_{k}\right)}{2}-A_{k}\right)\left(\frac{\left(A_{k}+B_{k}+C_{k}\right)}{2}-B_{k}\right)\left(\frac{\left(A_{k}+B_{k}+C_{k}\right)}{2}-C_{k}\right)\right)}}\]

Then we can calculate the position.

\[X_{k}=\frac{\left(a.x^{2}+a.y^{2}\right)\left(c.y-b.y\right)+\left(b.x^{2}+b.y^{2}\right)\left(a.y-c.y\right)+\left(c.x^{2}+c.y^{2}\right)\left(b.y-a.y\right)}{2\left(\left(b.x-a.x\right)\left(a.y-c.y\right)-\left(a.x-c.x\right)\left(b.y-a.y\right)\right)}\] \[Y_{k}=\frac{\left(a.x^{2}+a.y^{2}\right)\left(b.x-c.x\right)+\left(b.x^{2}+b.y^{2}\right)\left(c.x-a.x\right)+\left(c.x^{2}+c.y^{2}\right)\left(a.x-b.x\right)}{2\left(\left(b.x-a.x\right)\left(a.y-c.y\right)-\left(a.x-c.x\right)\left(b.y-a.y\right)\right)}\]

(You could define this as one variable but this is far more readable.)

Now we can display the circumcircle.

\[\left(x-X_{k}\right)^{2}+\left(y-Y_{k}\right)^{2}=r_{k}^{2}\]
